Streamboxd

Java (Quarkus) • React • Redis • PostgreSQL • Docker • Gitlab CI/CD

Automated Workflows • Scalable Architecture • Cloud-Ready Deployment

A fully featured and optimized movie streaming platform I built end-to-end as a high-level personal project. The goal was to create a solution that manages a complete movie catalog, automatically retrieves content from external sources, prepares it for online viewing, and delivers a smooth streaming experience to users.

The platform automatically handles processing from start to finish and keeps users updated in real time when content is being prepared or becomes available. Everything runs on a professional-grade, modular architecture with secure infrastructure and fully automated, efficient deployment and updates.

Ailcloud

Java (Quarkus) • React • PostgreSQL

Project Conception & Planning • Full-Stack Teamwork

Designed as a collaborative file management platform for organizations, this project was built end-to-end as a school project in team that we decided to push far beyond the original scope.

It provides secure authentication, role-based access, and structured workspaces where teams can manage both files and members. Files can be created, versioned, edited directly through an integrated editor, and deleted with strong access control. The platform also includes activity logging across users and projects. The interface was designed in Figma and built with React and Tailwind to deliver a clean, responsive SaaS-style experience.
